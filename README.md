# MVVMi-Boilerplate
Android MVVMi Boilerplate using Architecture Component

![diagram](https://github.com/RazibKani/MVVMi-Boilerplate/blob/master/android_mvvmi_diagram.png)

Libraries:
* Support Library
* Architecture Component
* Koin (Dependency Injection)
* Retrofit
* Room
* RxJava
* RxAndroid
* Anko
* Picasso
